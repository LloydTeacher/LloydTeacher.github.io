What is Software Testing?

Software Testing

is the process used to identify the correctness, completeness and quality of developed computer software

This process includes many different activities

  • test planning
  • analyzing
  • designing and implementing tests
  • reporting test progress and results
  • evaluating the quality of a test object

Those activities are parts of the Software Testing Life Cycle (STLC)

In order to improve the efficiency, improve the quality, automate and increase reliability of those activities

We use Testing Tools

Software Testing Tool

is a product that supports one or more test activities

  1. Requirements
  2. Planning
  3. Analysis
  4. Design
  5. Implementation
  6. Execution
  7. Completion

Such tools include:

  • Tools that assists in testing
    • Spreadsheet
    • Source code management Tool
  • Tools which helps to manage requirements, defects
    • Test management tools
  • Tools that are directly used in testing
    • Test execution tools
  • Tools that are used for investigation
