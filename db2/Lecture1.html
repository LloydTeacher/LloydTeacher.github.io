What is SQL?

SQL stand for Structured Query Language,
which is a computer language for storing, manipulating and retrieving data stored in relational database.

What are some SQL commands?

SQL commands categories

Data manipulation language

regroup commands which affect only the rows
  • Insert
  • Update
  • Delete

Data Query Language

Contains one command which capture Tables, columns and rows
  • Select

Data Definition Language

regroup commands which affect the entire table and the columns
  • CREATE
  • ALTER
  • DROP
