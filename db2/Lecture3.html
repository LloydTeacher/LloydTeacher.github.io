Data Definition Language

Alter Statements

Add a new column named gender
with a VARCHAR datatype
in a table named Persons

 ALTER TABLE Persons
    ADD COLUMN Gender VARCHAR(100);

Add a new column named age
with a integer datatype and the Default value of 1
in a table named Persons

 ALTER TABLE Persons
    ADD  age INT DEFAULT 1;

Modify a previous column named Birthday
with a date and time datatype
and the Default value of the current time
in a table named Persons

 ALTER TABLE Persons
    MODIFY COLUMN Birthday DATETIME
    DEFAULT CURRENT_TIMESTAMP;

Modify a previous column named PersonID
with an integer datatype
as a Primary key
in a table named Persons

 ALTER TABLE Persons MODIFY
    COLUMN PersonID INT PRIMARY KEY;

Modify a Primary key column named PersonID
to add the constraint auto increment
in a table named Persons

 ALTER TABLE Persons
    MODIFY AUTO_INCREMENT;

Modify a Primary key column named PersonID
to start the value of the auto increment to 24
in a table named Persons

 ALTER TABLE Persons
    MODIFY AUTO_INCREMENT=24;

Remove the column Height
in a table named Persons

 ALTER TABLE Persons
    DROP COLUMN Height;
