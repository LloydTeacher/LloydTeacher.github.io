What is a Relational Database Management System (RDBMS)?

Relational Database Management System (RDBMS)
A relational database management system (RDBMS) is a program/software that allows you to create, update, and administer a relational database.
Relational Database Management System (RDBMS)
Many relational database systems have an option of using the SQL (Structured Query Language) for querying and maintaining the database.

Install the MySQL RDBMS

There are many ways to install MySQL.

We will use the software bundle call XAMPP

Install XAMPP

  1. Click on this link XAMPP download
  2. Download the latest version for your operating system
  3. Run The installer

XAMPP Control Panel

  1. Activate MySQL
  2. Activate Apache

Launch phpmyadmin

Navigate to http://localhost/phpmyadmin

Create a new database or schema 

 CREATE DATABASE Country;

list all databases or schemas 

 SHOW DATABASES;
