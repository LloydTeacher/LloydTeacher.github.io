What is a Relational Database Management System (RDBMS)?

Relational Database Management System (RDBMS)
A relational database management system (RDBMS) is a program/software that allows you to create, update, and administer a relational database.
Relational Database Management System (RDBMS)
Many relational database systems have an option of using the SQL (Structured Query Language) for querying and maintaining the database.

Install the MySQL RDBMS

There are many ways to install MySQL.

We will use the software bundle call XAMPP

Install XAMPP

  1. Click on this link XAMPP download

  3. Run The installer

XAMPP Control Panel

  1. Activate MySQL
  2. Activate Apache

Launch phpmyadmin

Navigate to http://localhost/phpmyadmin

Create a new database or schema

 CREATE DATABASE Country;

list all databases or schemas 

 SHOW DATABASES;

Create a new database or schema
if it was not created previously

 CREATE DATABASE IF NOT EXISTS Country;

Remove a database or schema 

DROP DATABASE Country;

Remove database or schema
if it was created previously

 DROP DATABASE IF EXISTS Country;

Create a new table or entity

CREATE TABLE Persons (
    PersonID INT,
    FullName VARCHAR(25),
    Birthday DATE,
    height DECIMAL
);

list all tables or entities

 SHOW TABLES;

Create a new table or entity
if it was not created previously

 CREATE TABLE IF NOT EXISTS Persons (
    PersonsID INT,
    FullName VARCHAR(25),
    Birthday DATE,
    height DECIMAL
);

Remove a table or entity

DROP TABLE Persons;

Remove table or entity
if it was created previously

 DROP TABLE IF EXISTS Persons;
